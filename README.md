# IT1214-Student-Grade-System

## Project Purpose

This project is a Java Console-Based Student Grade Management System.

The system allows teachers to:

- Add student records
- Display all student records
- Search students by Student ID
- Calculate average marks of students

The project demonstrates Object-Oriented Programming (OOP) concepts and GitHub version control practices.

## How to Run

1. Compile the Java files

javac StudentGradeManagement.java

2. Run the program

java StudentGradeManagement

3. Use the menu options to manage student records.

## Technologies Used

- Java
- OOP Concepts
- GitHub Version Control
